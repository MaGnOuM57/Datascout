# 🎯 GUIDE PROSPECTION ARTISANS LUXEMBOURG

## ⚠️ CONTEXTE
Les annuaires professionnels luxembourgeois (Editus, WLW, etc.) bloquent le scraping automatique.  
**Solution** : Prospection manuelle ciblée + outils alternatifs.

---

## 🔍 MÉTHODES DE RECHERCHE EFFICACES

### 1. **GOOGLE MAPS** (Méthode #1 — La plus efficace)
**Recherche** : "couvreur luxembourg" / "toiture luxembourg"  
**Process** :
1. Ouvrir Google Maps
2. Chercher "couvreur luxembourg"
3. Filtrer : ⭐ >4.0 + >10 avis
4. Noter : Nom + Tel + Site web + Avis
5. Viser : 15-20 artisans

**Critères sélection :**
- ✅ Note >4.0/5
- ✅ >10 avis (entreprise établie)
- ✅ Photos chantiers haut de gamme
- ✅ Site web professionnel
- ❌ Éviter : artisans solo, <3 avis, pas de photos

---

### 2. **LINKEDIN** (Méthode #2 — Ciblage précis)
**Recherche** : "gérant couverture luxembourg" / "directeur toiture luxembourg"  
**Process** :
1. Recherche LinkedIn : "gérant couverture luxembourg"
2. Filtrer : Localisation Luxembourg + Titre "Gérant" ou "Directeur"
3. Vérifier profil entreprise (3-15 salariés = taille optimale)
4. Message LinkedIn ou extraction email via Hunter.io

**Avantage** : Contact décideur direct (pas secrétariat)

---

### 3. **PAGES JAUNES LUXEMBOURG (Editus.lu)** (Méthode #3 — Manuel)
**URL** : https://www.editus.lu/fr/recherche/toiture-luxembourg  
**Process** :
1. Ouvrir Editus.lu dans navigateur
2. Chercher "toiture" ou "couverture"
3. Copier manuellement : Nom + Tel + Adresse
4. Vérifier site web de chaque artisan
5. Créer liste Excel/Google Sheets

**Note** : Scraping bloqué → extraction manuelle nécessaire

---

### 4. **CHAMBRE DES MÉTIERS LUXEMBOURG** (Méthode #4 — Officiel)
**URL** : https://www.cdm.lu/  
**Process** :
1. Chercher annuaire membres
2. Catégorie "Couverture" / "Toiture"
3. Extraire contacts membres certifiés
4. Priorité : entreprises labellisées/certifiées

**Avantage** : Artisans certifiés = qualité garantie

---

### 5. **RÉSEAUX SOCIAUX** (Méthode #5 — Facebook/Instagram)
**Recherche** : "couvreur luxembourg" sur Facebook  
**Process** :
1. Facebook : Pages entreprises locales
2. Instagram : Hashtag #toitureluxembourg #couvreurlux
3. Identifier artisans actifs (posts réguliers)
4. Contact via Messenger/DM

**Avantage** : Artisans jeunes/digitaux = plus réactifs

## 📞 SCRIPT D'APPEL COMMERCIAL

**Introduction (15 sec) :**
> "Bonjour, je suis [Prénom] d'ApporteurCash. Je travaille avec des propriétaires luxembourgeois qui cherchent des artisans qualifiés. Vous avez 2 minutes ?"

**Pitch (30 sec) :**
> "J'ai actuellement 6 propriétés au Luxembourg (budget 20-50k€ chacune) avec des besoins toiture immédiats. Propriétaires solvables, vente en cours donc urgence haute. Je vends ces leads qualifiés à 80€ pièce. Pour que vous testiez la qualité, je vous envoie 1 lead gratuit en démo. Intéressé ?"

**Réponse objections :**
- _"J'ai déjà du travail"_ → "Parfait ! Ces leads sont ultra-qualifiés (25-50k€). Vous sélectionnez seulement ce qui vous intéresse."
- _"80€ c'est cher"_ → "Un chantier à 30k€, vous faites 8-10k€ de marge. 80€ = 0.3% du CA. C'est le moins cher du marché pour du haut de gamme."
- _"Garantie ?"_ → "Taux conversion 85%. Si ça ne marche pas, je vous rembourse 50%."

**Closing (15 sec) :**
> "Ok, je vous envoie le lead gratuit par email/WhatsApp dans l'heure. Vous validez la qualité, et si ça vous plaît, on discute des suivants. Votre email/WhatsApp ?"
