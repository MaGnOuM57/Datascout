# Script Téléphone - Lead Gen BTP

## Ouverture (10 secondes max)

```
Bonjour, [Nom] de [Entreprise]?

[Prénom] à l'appareil. Je génère des leads pour artisans BTP Luxembourg.

Vous avez 30 secondes?
```

**Si OUI:**

## Pitch (30 secondes)

```
Concrètement:
• Je trouve propriétaires/entreprises qui ont besoin de [votre service]
• Je valide contacts (tél + email)
• Vous livre un CSV avec notes

Pricing: 50€ contact basique, 100-150€ si projet actif confirmé.

Intéressé par échantillon gratuit (3 leads) pour voir qualité?
```

## Objections

### "Pas besoin, j'ai déjà du boulot"
```
Super! Et pour dans 2-3 mois? Je peux vous faire une réserve.
Ou je peux trouver des gros projets (>50k€) uniquement?
```

### "Trop cher"
```
Combien vous facturez un chantier en moyenne?
[Réponse: ex 5000€]
Donc 50€ pour un lead = 1% de votre CA potentiel.
Et si lead pas bon, je rembourse. Zéro risque.
```

### "Comment vous les trouvez?"
```
Scraping intelligent:
• Google Maps (propriétés haut de gamme)
• Permis de construire Luxembourg
• Nouvelles entreprises
• Validation manuelle avant livraison

Vous voulez un exemple gratuit pour voir?
```

### "Je vais réfléchir"
```
Parfait. Je vous envoie quand-même 3 leads gratuits par email?
Comme ça vous pouvez voir la qualité, pas d'engagement.
Votre email?
```

## Closer

```
OK super. Je vous prépare ça.

Pour la suite, si qualité OK:
• Batch 5 leads: 250€
• Batch 10 leads: 450€ (-10%)
• Abonnement 20 leads/mois: 800€ (-20%)

Je vous envoie l'échantillon d'ici 2h.
Vous me confirmez votre email?

[Noter email + entreprise + spécialité]

Parfait, à tout de suite!
```

## Timing

- **Meilleurs horaires**: 8h-10h, 17h-19h (avant/après chantiers)
- **Éviter**: 12h-14h (pause déjeuner)
- **Durée cible**: 2-3 min max par appel
- **Volume**: 20 appels/heure realistic
