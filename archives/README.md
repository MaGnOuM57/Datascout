# Archives (PARA System)

**Archives** = Inactive items from Projects, Areas, or Resources

## Purpose

Keep completed or inactive work out of the way but accessible.

## Structure

```
archives/
├── projects/
│   └── YYYY-MM-completed-project-name/
├── areas/
│   └── deprecated-area/
└── resources/
    └── outdated-info/
```

*Archive regularly to keep active workspace clean.*
