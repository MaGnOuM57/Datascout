# Knowledge Graph System

## Structure

This directory contains the knowledge graph layer - structured information, concepts, and relationships.

### Sub-directories

- **concepts/**: Core concepts, definitions, mental models
- **relationships/**: Mappings between concepts, people, projects
- **entities/**: Key entities (people, companies, tools, services)

### Usage

The knowledge graph complements:
- **MEMORY.md**: Long-term curated memory (high-level insights)
- **memory/YYYY-MM-DD.md**: Daily raw logs (chronological events)
- **knowledge/**: Structured relationships and concepts (queryable knowledge)

### Examples

```
knowledge/
├── concepts/
│   ├── lead-generation.md
│   ├── btp-market-luxembourg.md
│   └── cash-business-model.md
├── entities/
│   ├── tools.md
│   └── competitors.md
└── relationships/
    └── market-opportunity-map.md
```
