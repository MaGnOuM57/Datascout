## 🎯 Guide d'Utilisation Recommandé

### Étape 1 : Priorisation (30 min)
1. Filtrez les restaurants par région (votre zone de livraison)
2. Triez par score opportunité (commencez par 9.0+)
3. Vérifiez compatibilité type cuisine / votre produit

### Étape 2 : Recherche Complémentaire (1h)
1. Consultez le site web de chaque restaurant ciblé
2. Regardez leur carte actuelle (produits similaires ?)
3. Lisez les avis clients (mentions qualité produits ?)
4. Trouvez le nom du chef/responsable achats (LinkedIn)

### Étape 3 : Premier Contact (2-3h)
**Méthode recommandée** : Email + relance téléphonique J+2
- Jour 1 : Envoi email personnalisé (template ci-dessus)
- Jour 3 : Appel téléphonique si pas de réponse
- Jour 5 : Second email si toujours pas de réponse

### Étape 4 : Suivi & Conversion (en continu)
- Échantillon gratuit = taux conversion 30-40%
- Première commande = prix réduit -10%
- Fidélisation = service impeccable + régularité

**Taux de conversion estimé** : 10-15% (10-15 clients sur 100 prospects)

## 🔍 Critères de Scoring (0-10)

Chaque restaurant a été évalué selon :

1. **Budget estimé** (0-2 pts) - Basé sur standing + prix moyens
2. **Positionnement qualité** (0-2 pts) - Bio/terroir affiché ?
3. **Mentions "local/bio"** (0-2 pts) - Site web + carte + réseaux
4. **Avis clients** (0-2 pts) - Focus qualité produits
5. **Étoiles Michelin / Labels** (0-2 pts) - Certifications qualité

**Exemple** :
- L'Arpège : Budget 2/2 + Qualité 2/2 + Bio 2/2 + Avis 2/2 + 3* Michelin 2/2 = **9.5/10**
