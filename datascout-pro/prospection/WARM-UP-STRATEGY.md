## 🛡️ Checklist Anti-Spam (AVANT chaque batch)

### Contenu Email
- [ ] Personnalisation réelle (prénom + entreprise + produit spécifique)
- [ ] Ratio texte/liens > 90% texte
- [ ] Maximum 1 lien (vers landing page)
- [ ] Pas de spam words ("gratuit", "argent", "garantie" en majuscules)
- [ ] Désabonnement clair ("STOP" = unsubscribe)
- [ ] Signature complète (nom, entreprise, adresse physique idéalement)

### Technique
- [ ] SPF + DKIM + DMARC tous verts (✅)
- [ ] Domaine vérifié Resend
- [ ] Delay 3-5 min entre chaque email (pas burst)
- [ ] Plain text (pas HTML riche = spam trigger)
- [ ] From name professionnel ("DataScout Pro" vs "contact@")

### Monitoring
- [ ] Check mail-tester.com score (≥9/10) avant batch
- [ ] Monitor bounce rate live (<3% = OK, >5% = stop)
- [ ] Check spam complaints (1 seule = analyser urgence)
- [ ] Vérifier inbox placement (Gmail, Outlook, ProtonMail)
